- 模拟信号和系统

### 2. 性质

1. 周期性
   1. 序列的傅里叶变换是频率w的周期函数, 周期为$$2\pi$$
   2. X($$e^{jw}$$)可展成傅里叶级数， x(n)是其系数
   3. ω＝0,±2π,±4π…时， X(ejω)表示信号的直流分量，在ω＝(2M＋1)π时， X($$e^{jw}$$)是最高的频率分量。一般只分析信号在一个周期的FT，即：0到2π或者-π到π，其中π为最高频率.**最高频率分量$$e^{jn{\pi}}$$**
2. 线性
   1. ![image-20221207130651607](C:\Users\zZOMZz\AppData\Roaming\Typora\typora-user-images\image-20221207130651607.png)
3. 时移和频移
   1. <img src="C:\Users\zZOMZz\AppData\Roaming\Typora\typora-user-images\image-20221207130818566.png" alt="image-20221207130818566" style="zoom:50%;" />

4. **对称性**: 
   1. 共轭对称序列: 实部是偶函数, 虚部是奇函数
   2. 共轭反对称序列: 实部是奇函数, 虚部是偶函数
   3. FT的实部具有共轭对称性, FT的虚部具有共轭反对称性
   4. x(n)=$$x_e(n)$$+$$x_o(n)$$ , $$x_e(n) = \frac{1}{2}[x(n) + x^*(-n)]$$,  $$x_o(n) = \frac{1}{2}[x(n) - x^*(-n)]$$
   5. **根据实因果序列特性，h(n)=he(n)U+(n)**

5. 时域卷积定理:
   1. 若$$y(n) = x(n) * h(n)$$则$$Y(w) = X(w)H(w)$$
6. 频域卷积定理:
   1. 若$$y(n) = x(n)h(n)$$则$$Y(w) = \frac{1}{2{\pi}}X(w)*H(w)$$

## 2.6 利用Z变换分析信号和系统的频域特性

### 1. 传输函数与系统函数

设系统的初始状态为零,输入单位脉冲序列$$\delta(n)$$,输出端得到的信号,称为系统的**单位脉冲响应h(n)**

![image-20221207154337753](C:\Users\zZOMZz\AppData\Roaming\Typora\typora-user-images\image-20221207154337753.png)

单位脉冲信号的物理意义:  输出增益: $$|H(jw)|$$, 虚部$$e^{j\phi(w)} $$相移: $$\phi(w) < 0$$ 

### 2. 用系统函数的极点分布判断系统因果性和稳定性

#### 1. 因果系统

- n < 0 时, h(n) = 0
- H(z)的收敛域不含0,含有$$\infty$$点, 收敛域在某圆外

#### 2.稳定系统

- $$\sum_{n = -\infty}^{\infty}{|h(n)|} < \infty$$
- H(z)的收敛域包含单位圆

#### 3. 系统因果且稳定

- 收敛域两个要求都要符合,要包含单位圆和$$\infty$$, 即 0 < r < 1
